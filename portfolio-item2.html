Web developer

Algonquin College - WoodYouKnow (Twintimes)

Jan 2021 - Dec 2021

This WoodYouKnow woodworking desktop application is designed to serve as a repository of information for either professional or hobbyist woodworkers who would want to access information regarding prices of tools across many different retailers, display graphs with information pertinent to cost variances, export data to excel format, and have currency conversion options. The application will also allow the user to access information regarding the safe use of equipment such as user manuals.

This app uses Electron, which helps us build the app like a website using HTML, CSS, and JavaScript.
